## рЯФІ Shared Workflows

This repo provides reusable GitHub Actions workflows that every product repo's `release.yml` calls at tag time:

- `.github/workflows/release-book.yml` вАФ compiles a styled PDF release book from the caller repo's markdown docs (pandoc + XeLaTeX).
- `.github/workflows/release-video.yml` вАФ renders a silent MP4 screencast of the caller repo's marketing `index.html` (Playwright + ffmpeg).
- `.github/workflows/release-dashboard.yml` вАФ refreshes the release dashboard table below.

Product repos call them like:

```yaml
release-book:
  uses: AmericanGroupLLC/AmericanGroupLLC/.github/workflows/release-book.yml@main
  with:
    app-name: ${{ needs.setup.outputs.app-name }}
    version: ${{ needs.setup.outputs.version }}
    brand-color: ${{ needs.setup.outputs.brand-color }}
    docs: ${{ needs.setup.outputs.docs }}
  secrets: inherit
```

Cross-repo `uses:` requires a one-time settings change in this repo documented in `SECRETS.md` ¬І0.

See also:

- [`SECRETS.md`](SECRETS.md) вАФ every secret consumed and how to set it.
- [`templates/`](templates/) вАФ pandoc + video templates.
- [`scripts/`](scripts/) вАФ local CLI mirrors of the shared workflows (handy for dry-runs).
